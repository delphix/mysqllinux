    DXI 1.3.1 Release Notes¶

    To install dxi, refer to Getting started

    New Features¶

    • dxi Dockerfile
    • dxi Docker Hub image

    (At this point, dxi docker container does not support the encryption of config file dxtools.conf. This support will be added in a future release)

    Requirements¶

    • Python versions 3.7 or above
    • Delphix Engine 5.3.9 or above
    • macOS 10.15+
    • Windows 10
    • RHEL 7,8

    Future releases may add support for additional OS platforms and Delphix Engine versions.

    Questions?¶

    If you have questions, bugs or feature requests reach out to us via email
